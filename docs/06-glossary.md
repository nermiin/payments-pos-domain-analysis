# Glossary — Payments & POS

| Term | Definition |
|------|------------|
| **Authorization** | Issuer approval to reserve funds up to amount |
| **Capture** | Confirmation to clear and settle funds |
| **Void** | Cancellation of uncaptured authorization |
| **MID** | Merchant ID — acquirer-assigned identifier |
| **TID** | Terminal ID |
| **MCC** | Merchant Category Code |
| **ISO 8583** | Legacy financial transaction messaging standard |
| **P2PE** | Point-to-Point Encryption — PCI scope reduction for POS |
| **Store-and-forward** | Offline terminal queueing transactions until online |
| **Idempotency Key** | Client-supplied unique key preventing duplicate operations |
| **Auth code** | Issuer reference for approved authorization |
| **Clearing** | Exchange of txn data post-authorization |
| **Settlement** | Funds transfer to merchant bank account |
